# Sustify Backend

Fast, type-safe NestJS starter focused on clean validation and modular structure.

## Highlights
- NestJS 11 with TypeScript, ready for prod builds.
- Validation utilities for enums, payload shape, email, and phone numbers.
- Modular layout (commons, validators, iam) to keep domains separated.
- Prettier + ESLint already wired in for consistent code.

## Quick Start
```bash
npm install
npm run start:dev
```

## Testing
```bash
npm test
```

## Project Scripts
- `npm run lint` — fix lint issues.
- `npm run build` — compile to `dist/`.
- `npm run start:prod` — run the built app.
